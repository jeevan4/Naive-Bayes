To Install 64 bit NUMPY
========================
Download the .whl file from this link. http://www.lfd.uci.edu/~gohlke/pythonlibs/
Downloaded file has name "numpy-1.9.2+mkl-cp34-none-win_amd64.whl"

Go to command prompt and install the .whl using this command in python folder.
C:/Python34> python -m pip install .whl filepath

In my case the command is : python -m pip install -U C:\Users\Jeevan\Downloads\numpy-1.9.2+mkl-cp34-none-win_amd64.whl
s
If it says, it is already installed, u can upgrade it, then use

python -m pip install -U C:\Users\Jeevan\Downloads\numpy-1.9.2rc1+mkl-cp34-none-win32.whl
